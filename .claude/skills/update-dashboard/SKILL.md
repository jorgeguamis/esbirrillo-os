# Update Dashboard Skill

Regenerate the Dashboard with current data from daily notes, context files, and Life OS system.

---

## When to Use

- Weekly (every Sunday or Monday morning)
- After completing monthly/quarterly reviews
- When priorities or focus areas change significantly
- Before important planning sessions

**Trigger:** `/update-dashboard`

---

## Data Sources

### Required

**Daily Notes:**
- Last 7 days: `02. Areas/personal/diario/01. Daily/DD-MM-YYYY.md`
- Extract: Wins, energy drains, what moved the needle, TOP 2 progress

**Context Files (7 total):**
- `02. Areas/personal/Personal.md` - Current priorities, Life Map ratings
- `02. Areas/business_primary/{{BUSINESS_PRIMARY}}.md` - Current focus, priorities, metrics
- `02. Areas/business_secondary/{{BUSINESS_SECONDARY}}.md` - Current focus, priorities, metrics
- `02. Areas/salud/Health.md` - Health goals, habits
- `02. Areas/aprendizaje/Learning.md` - Learning focus
- `02. Areas/finanzas/Finances.md` - Financial context
- `02. Areas/investigacion/Research.md` - Research projects

**Monthly Review:**
- Most recent: `02. Areas/personal/diario/03. Monthly/YYYY-MM.md`
- Extract: Life pillar ratings, key metrics

**Task Brainstorm:**
- `_Tasks.md` - Pending tasks by urgency level

**North Star:**
- `02. Areas/personal/North Star.md` - Strategic orientation

---

## Process

### Step 1: Read Source Data

**From last 7 daily notes:**
```
For each day (last 7):
  - Read daily note
  - Extract: Wins, Energy drains, TOP 2 completion
  - Track energy levels (1-10)
```

**From all context files:**
```
Read each context file:
  - Current top 3 priorities
  - Current focus area
  - Business metrics (if applicable)
```

**From most recent monthly review:**
```
Extract Life Map ratings:
  - Career ({{BUSINESS_PRIMARY}}) rating
  - Career ({{BUSINESS_SECONDARY}}) rating
  - Relationships rating
  - Health rating
  - Finances rating
  - Learning rating
```

---

### Step 2: Generate Dashboard Sections

**AUTO-GENERATED Sections (80-90%):**

1. **This Week Status**
   ```
   - Wins this week: [Top 3 from daily notes]
   - Energy drains: [Common patterns from daily notes]
   - What moved the needle: [TOP 2 tasks completed that align with priorities]
   ```

2. **Active Areas**
   ```
   For each area (Business-360, Business-{{BUSINESS_SECONDARY}}, Personal):
     - Current focus: [from context.md]
     - Next milestone: [from projects or goals]
   ```

3. **Life Pillars Quick Check**
   ```
   | Pillar | Rating | Status |
   From most recent monthly review Life Map
   Calculate overall average
   Identify pillar to prioritize (lowest rating + highest priority)
   ```

4. **Key Metrics**
   ```
   {{BUSINESS_PRIMARY}}: [Revenue, clients, pipeline from context]
   {{BUSINESS_SECONDARY}}: [Students, workshops, revenue from context]
   Personal: [Health score, energy average, deep work days from daily notes]
   ```

5. **This Month Focus**
   ```
   From monthly review + context files:
     - Key goals for month
     - Progress indicators
   ```

**MANUAL Sections (10-20%) - DO NOT OVERWRITE:**
- Today's Focus (Quick Win, TOP 2)
- Energy Check (current)
- Vision Reminders (one-word theme, 3-year vision, commitment)

---

### Step 3: Update Dashboard File

**Location:** `_Dashboard.md`

**Process:**
1. Read current Dashboard
2. Identify AUTO-GENERATED sections (marked with `<!-- AUTO-GENERATED -->` comments)
3. Replace ONLY auto-generated content
4. Preserve ALL manual sections
5. Update "Last auto-updated" timestamp

**Format for auto-generated sections:**
```markdown
<!-- AUTO-GENERATED by /update-dashboard on YYYY-MM-DD HH:MM -->
[Generated content here]
<!-- END AUTO-GENERATED -->
```

Updated `_Dashboard.md` with:
